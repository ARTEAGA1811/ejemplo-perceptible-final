Pauta: Perceptible

Alternativas de texto

Se proporcionan alternativas de texto para cualquier contenido que no sea de texto para que se pueda cambiar a otras formas que las personas necesiten.

Por ejemplo, si usamos una imagen, una persona ciega o con visión reducida no la podrá ver, pero si añadimos un texto o descripción, se podrá utilizar herramientas de asistencia para que el texto pueda ser leído, cambiado a lenguaje braile, aumentar su tamaño de fuente, etc.

Instrucciones:

1. Debajo hay una imagen con el siguiente código (línea 65 del archivo index.html):

<figure>
  <img src="..." alt="..." />
</figure>

2. Agrega una alternativa de texto a la imagen. Pista: Existe una etiqueta llamada figcaption.

Medios Basados en el tiempo

  • Subtítulos (pregrabados)

La intención de este Criterio de Conformidad es permitir que las personas sordas o con dificultades auditivas vean presentaciones multimedia sincronizadas.

Por ejemplo, se añaden subtítulos a un video, una descripción de audio a una película, se realiza una transcripción a un video o audio, etc.

Adaptable

  • Información y relaciones

Se debe percibir una estructura y las relaciones a través de varias señales visuales

Ejemplos:
  1. Los encabezados suelen estar en una fuente más grande y en negrita
  2. Los párrafos están separados por una línea en blanco
  3. Las palabras que tienen un estatus especial se indican cambiando la familia de fuentes y/o poniéndolas en negrita, cursiva o subrayadas

Instrucciones:

1. Coloca en negrita a la frase: "el grado de facilidad de uso que tiene una página web para los visitantes que entran e interactúan con ella."

2. Coloca en cursiva la palabra: "usabilidad"

3. Subraya la palabra: "intuitiva"

La usabilidad web es el grado de facilidad de uso que tiene una página web para los visitantes que entran e interactúan con ella. Una web con una buena usabilidad es aquella que permite a los usuarios una interacción sencilla, intuitiva, agradable y segura.

Distingible

  • Contraste

La intención de este Criterio de Conformidad es proporcionar suficiente contraste entre el texto y su fondo para que pueda ser leído por personas con una visión moderadamente baja.

Instrucciones:

1. Ingresa a Contrast Finder para encontrar colores con buen contraste.

2. El color del fondo es rojo (#FF0000), el color actual del texto es amarillo (#FFFF00). Busca un color para el texto que tenga un contraste de mínimo 4.5.

3. Ingresa a la hoja de estilos style.css, al final se encuentra la clase color_del_texto, en esta clase cambia el color del texto.
